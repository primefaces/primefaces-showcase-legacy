Layout - Element * Deprecated

Layouts can also be created at element level. Example below is a simple split panel implementation. West Center 
<p:layout style="min-width:400px;min-height:200px;">
    <p:layoutUnit position="west" resizable="true" size="100" minSize="40" maxSize="200">
        West
    </p:layoutUnit>

    <p:layoutUnit position="center">
        Center
    </p:layoutUnit>
</p:layout>
